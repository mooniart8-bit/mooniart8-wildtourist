Книга памяти (Memory Books)

ВАЖНО! Версия вашей SillyTavern обязательно должна быть выше 13.1.5. Проверьте свою версию перед установкой.

Установка расширения

Пошаговая инструкция установки и настройки Memory Books:

1. Скопируйте эту ссылку (не переходите по ней):
https://github.com/aikohanasaki/SillyTavern-MemoryBooks
2. Вставьте её в менеджер расширений SillyTavern.
3. Дождитесь уведомления об успешной установке.
4. Перезагрузите страницу (для ПК) или перезайдите в сессию (для Termux).

Настройка создания записей

Для Memory Books понадобится отдельный чистый Лорбук (Lorebook), куда будут записываться события вашего чата. Этот Лорбук необходимо прикрепить к чату вашего выбранного бота. Если вы делаете ветку или чекпоинт, Memory Books надо переключить на другой Лорбук.

Рекомендуемые настройки:

  • Выставьте создание записи каждые 20 сообщений.
  • Не ставьте значение выше 50 записей — система выдаст ошибку.
  • Сделайте отметку сцен согласно выбранному количеству записей.

Выбор провайдера (API)

В Memory Books можно выбрать отдельный профиль соединения для создания записей. По умолчанию используются настройки вашего текущего подключения в SillyTavern.

Внимание: Gemini 2.5 pro не рекомендуется для Memory Books из-за постоянной цензуры. Настройте профиль на другого провайдера.

Для смены провайдера:

1. Нажмите "Новый профиль" в настройках Memory Books.
2. Выберите бесплатную модель, которая не является вашим основным API.
3. Рекомендуемые прокси: Routeway, MegaLLM, OpenRouter.
Примечание: ELECTRON HUB использовать не советую из-за маленького контекста моделей, но для небольших постов подойдет.

Промпты и локализация

Стандартный Prompt для создания записей написан на русском языке, а значит и записи будут генерироваться на русском. Если вы ведете ролевую игру на английском, вам придется отредактировать prompt в настройках расширения и перевести его.

Генерация воспоминаний

После нажатия кнопки «Создать воспоминания», пойдет загрузка.

  • Если плашка становится желтой и выдает ошибку — выбранная модель перегружена или прокси провайдера упал.
  • Если вас устраивает сгенерированная запись — добавляйте её в Лорбук. Если нет — отредактируйте текст под себя вручную.
