Установка SillyTavern на ПК

ВАЖНО: Перед установкой убедитесь, что у вас установлен Node.js (рекомендуется версия LTS).

Шаг 1: Скачивание и установка

Здесь будет располагаться текст с инструкцией по установке через GitHub Desktop или скачиванию ZIP-архива.

Шаг 2: Запуск

1. Откройте папку с установленной SillyTavern.
2. Запустите файл Start.bat.
3. Дождитесь, пока в консоли не появится локальный адрес (обычно http://localhost:8000).
