Установка SillyTavern на ПК.

ВАЖНО: Возможно понадобиться VPN при установки. Сначала попробуйте без него - не выходит - включайте.
Я буду рассматривать только установку через Git, потому что это самая быстрая и стабильная версия SillyTavern. Есть способ через лаунчер, но его не сильно рекомендуют. Выявляются много приколов после.

Шаг 1: Установка Git.

1. Переходите на сайт Git и устанавливайте стандартную версию на Windows гит - https://git-scm.com/install/windows
2. Запустите файл Start.bat.
3. Дождитесь, пока в консоли не появится локальный адрес (обычно http://localhost:8000).

Шаг 2: Запуск

1. Откройте папку с установленной SillyTavern.
2. Запустите файл Start.bat.
3. Дождитесь, пока в консоли не появится локальный адрес (обычно http://localhost:8000).
