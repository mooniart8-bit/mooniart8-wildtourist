Установка SillyTavern на ПК.

ВАЖНО: Возможно понадобиться VPN при установки. Сначала попробуйте без него - не выходит - включайте.
Я буду рассматривать только установку через Git, потому что это самая быстрая и стабильная версия SillyTavern. Есть способ через лаунчер, но его не сильно рекомендуют. Выявляются много приколов после.

Шаг 1: Установка Git.

1. Переходите на сайт Git и устанавливайте стандартную версию на Windows гит - скачать Git Windows Если будет при установки плашка аля "Добавить в Pathc" - добавляйте.

2. Переходите на сайт Node и устанавливайте версию v22.22.2 - скачать Node

ВАЖНО: Именно версию v.22.22.2 - потому что с ней не выходят ошибки связанная с версией Node.

3. На рабочем столе создайте пустую рандомную папку и назовите ее английскими буквами как хотите (ключевое слово АНГЛИЙСКИМИ БУКВАМИ - потому что у вас может выходить ошибка)

4. Заходите в свою пустую папку. Кликайте на путь и вводите cmd

5. Жмете Enter и появляется консоль.

6. В консоль вводите команду:
git clone https://github.com/SillyTavern/SillyTavern -b release
Нажимаете Enter и ждете окончания установки.

7. Ваша SillyTavern появится в папке после установки.

Шаг 2: Как запускать?

1. Заходите в папку SillyTavern которая у вас образовалась после установки. Листаете в самый низ.
2. Файл Start.bat запускает SillyTavern. Файл UpdateAndStart.bat обновляет и запускает SillyTavern. Я вам рекомендую все-таки обновлять иногда SillyTavern. Обновления выходят не особо часто.

3. Дождитесь, пока в консоли не появится локальный адрес (обычно http://localhost:8000). SillyTavern откроется в браузере. Предпочитаемые браузеры для SillyTavern - Google Chrome и Mozilla Firefox.
