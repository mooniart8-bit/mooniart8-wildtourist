Установка SillyTavern на Android

ВНИМАНИЕ: Для установки на Android мы используем приложение Termux. Скачивайте его только с F-Droid или Github, версия из Google Play не поддерживается!
Делаем все исключительно под VPN!
ВНИМАНИЕ:Также необходимо для каждого приложения, с которого вы будете запускать SillyTavern, дать разрешения:
1. Фоновый режим (он всегда должен работать в постоянном режиме)
2. Снять ограничение на энергосбережение.
3. Дать доступ к файлам.
4. Дать доступ к уведомлениям.
5. Если у вас все еще будет виснуть или лагать кнопки на начальном этапе настройки, значит необходимо вынести приложение через которое вы запускали SillyTavern в плавающее окно.

Официальная документация SillyTavern


1. Вставляем команду - termux-change-repo. Там где у вас символ "$", зажимаете пальцем, чтобы высветилось меню и нажимаете "Paste" - это вставка команды.

2. Выбираем зеркало.

3. Когда закончится загрузка зеркала, вставляете следующую команду - pkg update && pkg upgrade





4. Когда закончится, то вставляем следующую команду - pkg install git nodejs-lts nano


5. Когда закончится, то вставляем следующую команду - git clone https://github.com/SillyTavern/SillyTavern -b release

6. Когда закончится, то вставляем следующую команду - nano ~/.bashrc

7. У вас откроется страшная консолька, не пугаемся ее. Вставляем следующее:
# Update Termux packages alias pkgup="pkg update && pkg upgrade" #Start SillyTavern alias st='cd ~/SillyTavern && bash start.sh' # Update SillyTavern alias stup='cd ~/SillyTavern && git pull --rebase --autostash'





8. Когда закончится, то вставляем следующую команду - nano ~/.bashrc и нажимаете Enter. После вы сможете запускать таверну через команду "st"


ВАЖНО: Загрузка будет долгая. После будет открываться в браузере. Предпочтительно выбирать Google или Mozilla Firefox (они должны стоять по умолчанию, чтобы открывать таверну в этих браузерах).
Также полный видеогайд этого способа вы можете посмотреть по данному сообщению, если находитесь в нашем чате - Видеогайд.

Termux и команда установка.


ВАЖНО: Termux строго тот файл, который указан в посте. С другой версией Termux работать не будет.
1. После скачивания и установки Termux.apk из поста, вставляете команду - pkg update -y && apt upgrade -y -o Dpkg::Options::="--force-confold" && pkg install git nodejs -y && git clone https://github.com/SillyTavern/SillyTavern && cd SillyTavern && bash start.sh



2. Дальнейший запуск строго через команду - SillyTavern/start.sh

SillyTavern через ST-android.

Огромное спасибо тгк Гайды от Макса | SillyTavern за интересный находку. Основной пост тут - Пост об ST
1. Переходим на GitHuib - ST-android
2. Листаем вниз и выбираем "Releases", затем скачиваем файл - ST-android-0.4.0.apk и устанавливаем.


3. Перед открытием ST, необходимо приложению дать разрешение на фоновый режим и снять ограничение на энергосбережение.

4. Далее у вас всплывет прашка. Кликаете "Разрешить".

5. Далее у вас будет еще плашка. Кликаете "Set и Пропустить".


6. Переходим к установке. Кликаем "Manage ST".

7. Дальше кликаем "Load Branches and Tags" и выбираем ту версию, что указана в скрине. Устанавливаем ее.


8. Установка завершится, когда данное окно пропадет (их будет два).

9. Наверху у вас будет отображаться версия. На кнопку "Start" у вас запуск SillyTavern.

10. Пройдемся по еще важным аспектам данного метода установки. Как смотреть где у вас ошибки. На кнопку "Logs" у вас запуск ваши Логи. Там описываются ошибки.


11. Чтобы внести изменения в файл config, кликаем на кнопку "Edit ST config"

12. Чтобы внести изменения в системные файлы SillyTavern, придется скачивать бэкапы и импортировать их обратно через "Import Data", которая находится в Manage ST.
