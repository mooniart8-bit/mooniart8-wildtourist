Установка SillyTavern на Android

ВНИМАНИЕ: Для установки на Android мы используем приложение Termux. Скачивайте его только с F-Droid или Github, версия из Google Play не поддерживается!

Шаг 1: Скачивание с F-Droid или Github.

скачать с F-Droid


pkg update && pkg upgrade -y
pkg install git nodejs -y

Шаг 2: Клонирование репозитория и запуск

Здесь будет текст с командами клонирования GitHub репозитория и запуском `bash start.sh`.
